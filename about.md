![Jeroen Meppelink](/assets/img/jeroen.jpg)

Jeroen Meppelink is a **Lead Infrastructure Engineer** focused on End User Computing, Infrastructure as Code and DevOps at enterprise scale. He designs and runs Citrix platforms so they behave more like a product than a firefight — daily idempotent Ansible runs for desired state, pipelines for updates and certificate rotation, and self-service APIs so application teams can ship without filing tickets.

His guiding rule is simple: if a human is doing it twice, it belongs in code.

Outside work Jeroen is a Home Assistant hobbyist and open-source tinkerer, building custom integrations in Python and JavaScript. The most-used one is *ns-reisadvies*, a Dutch train travel advisor that runs on his home network. He writes here on [meppie.eu](https://meppie.eu) when something from the day job is worth sharing.

**Based in:** Hilversum, the Netherlands.
