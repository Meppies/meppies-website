## Live

### dev-standards-mcp
**Developer tool · open source · MCP**

A shared knowledge base of engineering standards, connected to every developer's AI tool via the Model Context Protocol. The AI reads the standards at the start of every session, so it writes code the way your team writes code — without anyone having to explain it first.

A fork-and-customise starter is on GitHub: [github.com/Meppies/develop-standards-starter](https://github.com/Meppies/develop-standards-starter). A guided setup walks you through filling it in for your own stack in one session.

Longer write-up: [From second brain to team brain](/blog/from-second-brain-to-team-brain/).

### ns-reisadvies
**Home Assistant integration · JavaScript + Python**

A Dutch train travel advisor that runs inside Home Assistant. Shows upcoming departures, delays and disruptions for any route, surfaced in Lovelace cards and automations. Source: [github.com/Meppies/fun-things-to-share](https://github.com/Meppies/fun-things-to-share).
